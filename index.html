为什么我又把博客搭建了起来？

发表于 2025-01-27 更新于 2025-02-06 分类于 说说 本文字数： 114 阅读时长 ≈ 1 分钟

现在各种媒体那么多，为什么我又把博客搭建来起来。主要是基于以下的思考：

  1. 媒体速度转换快，各种信息非常的分数，如果需要一个完全属于自己的一个阵地的话还得是博客，可以完成的记录自己想发的东西。
  2. 各个平台有审核机制，还是自己的博客最靠谱。

为什么版权越来越规矩，歌曲却越来越不火了

发表于 2025-01-26 更新于 2025-02-06 分类于 说说 本文字数： 140 阅读时长 ≈ 1 分钟

自从歌曲的版权被保护的越来越好之后，歌手成名越来越难了。

现代社会贵的不是这些歌曲，而是每个人的注意力资源，所以保护版权对歌手并不是好事情，反而让自己的歌曲传唱开，去走穴，开演唱会是更好的模式。

互联网也是类似的模式，用户使用免费，广告收费最终是成功的模式。当进入AI时代后会如何呢？
